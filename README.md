# TG1_Grupo2Leit52
🛒 Sistema de Carrinho de Compras

Aplicação web para gestão de carrinho de compras em loja de e-commerce, desenvolvida com arquitetura MVC (Model-View-Controller).

#  Descrição do Projeto

Este projeto foi desenvolvido como parte da disciplina ASI, com o objetivo de criar uma aplicação web funcional para gerenciamento de carrinho de compras, seguindo os princípios da arquitetura MVC e boas práticas de desenvolvimento web

##  Funcionalidades

- ✅ Adicionar Produtos - Cadastro de produtos com nome e preço
- ✅ Visualizar Lista de Produtos - Exibição de todos os produtos disponíveis
- ✅ Adicionar ao Carrinho - Inclusão de produtos no carrinho de compras
- ✅ Visualizar Carrinho - Listagem de itens adicionados ao carrinho
- ✅ Total de Itens - Contagem da quantidade total de produtos no carrinho
- ✅ Total em Valor - Somatório dos preços dos produtos no carrinho
- ✅ Encerrar Compra - Botão que reinicia o carrinho (limpa todos os itens)
- ✅ Persistência de Dados - Os dados são salvos no localStorage do navegador
- ✅ Validação de Preços- Aceita apenas números e ponto decimal

 🏗️ Arquitetura MVC

O projeto segue rigorosamente o padrão MVC (Model-View-Controller).

 Tecnologias Utilizadas

- HTML5 - Estrutura da aplicação
- CSS3 - Estilização e layout responsivo
- JavaScript  - Lógica da aplicação
- LocalStorage - Persistência de dados
- GitHub Pages - Hospedagem
